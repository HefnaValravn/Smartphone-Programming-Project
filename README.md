# Smartphone-Programming-Project

## Overview

My first mobile application project! This application aims to be a sort of "personal CV app" for my mid-semester assignment, and has a few features we're supposed to build.


## Prerequisites & Dependencies

To build and run this project, you will need the following:

*   **Java Development Kit (JDK):** Ensure you have a compatible version installed (e.g., JDK 8 or later).
*   **Android SDK:** Required for Android development, including platform tools, build tools, and SDK platforms.
*   **Android Studio:** The official IDE for Android development. Download and install from [https://developer.android.com/studio](https://developer.android.com/studio).
*   **Gradle:** The build automation tool used by Android Studio. Android Studio typically manages Gradle automatically.

## Installation & Setup Instructions

Follow these steps to install and set up the project:

1.  **Clone the Repository:**

    ```bash
    git clone https://github.com/HefnaValravn/Smartphone-Programming-Project.git
    ```

2.  **Open in Android Studio:**

    *   Launch Android Studio.
    *   Select "Open an Existing Project."
    *   Navigate to the cloned repository's directory and select the `SmartphoneProject` folder.

3.  **Sync Gradle:**

    *   Android Studio will prompt you to sync the Gradle files.  Click "Sync Now."
    *   Resolve any dependency issues that may arise.

4.  **Build the Project:**

    *   In Android Studio, go to "Build" > "Make Project" or "Build" > "Rebuild Project".

5.  **Run the Project:**

    *   Connect an Android device or start an Android emulator.
    *   Click the "Run" button (green play icon) in Android Studio.
    *   Select your connected device or emulator.
